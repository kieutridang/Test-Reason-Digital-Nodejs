# Starting Guide

## 1. Installation

Create `.env` file with content from `.env.example`

Run `yarn install`

## 2.1. Start the application

Run `yarn start`

## 2.2. Start in development mode

Run `yarn dev`

# Endpoints

1. Login: POST `/login`
2. Get all posts: GET `/posts`
3. Get post by id: GET `/posts/{id}`
